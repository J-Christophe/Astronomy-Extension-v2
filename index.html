Welcome to Astronomy-Extension Pages.

Astronomy-Extension repository is a set of SITools2's services that provides some access capabilities related to the astronomical field. This page describes the different services and how to set up them in SITools2.

Overview

The package is composed of:

  • server plugins providing new access capabilities
  • client extension called Module providing new graphical user interfaces.

Server plugins

This section gives a short overview of all server plugins that are available within the Astronomy-Extension repository

  • Data converters
  • Data filters
  • Access security filters for data storages
  • Dataset services
  • Application services

Module

This section gives a short overview of all modules that are available within the Astronomy-Extension repository

  • MIZAR (Module for Interactive visualiZation from Astronomical Repositories)
